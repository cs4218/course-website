[](#design-entities)Design Entities
===================================

The design entities characterize and help refer to different abstraction from the program. A list of design entities for SIMPLE programming language are as follows:

*   procedure
*   stmt
*   read (statement)
*   print (statement)
*   assign (statement)
*   call (statement)
*   while (statement)
*   if (statement)
*   variable
*   constant
